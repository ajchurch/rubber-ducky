# A7. Discover cryptography used in modern networks.

## WPA3 (WiFi Protected Access 3)

WPA3 improves upon the older WPA2 standard by replacing the vulnerable "four-way handshake" with a more secure cryptographic exchange called Simultaneous Authentication of Equals (SAE). While WPA2 allowed attackers to capture login data and attempt millions of password guesses offline, WPA3 requires an active, unique interaction for every single attempt, effectively neutralising brute-force attacks. Furthermore, WPA3 introduces forward secrecy, which ensures that even if a network password is compromised in the future, an attacker cannot use it to decrypt any data captured from previous sessions. By using Elliptic Curve Cryptography to verify the password without ever actually transmitting it, WPA3 provides a much more resilient "digital lock" than the Pre-Shared Key (PSK) system used in WPA2.
