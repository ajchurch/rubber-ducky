# A28. Implement a security solution of your choice and put it on your GitHub.

I have created a SUPER simple Password Strength Checker as my security solution. Essentially, it is what the title of it says, it checks the strength of your password that you input and gives it a score out of 100. 

It provides suggestions for ways to improve the password and it is all done locally through a python script via the command line.

This is a security solution because passwords are a core part of keeping accounts safe and secure, and this is a very simple way of verifying the strength of a password. Although it is not a replacement to something like a password manager which does it much quicker and better, this is purely just a proof of concept for the points of this portfolio.  

You can try it [here](https://github.com/ajchurch/password-strength-checker)

From the README.md about how to use it:
```bash
git clone https://github.com/ajchurch/password-strength-checker.git
cd password-strength-checker
python main.py
```
You'll be prompted to enter a password securely (input is hidden).

Or you can pass the password as an argument (for testing/scripting).
```
python main.py "MyP@ssword123!"
```
