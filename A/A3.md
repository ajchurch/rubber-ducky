# A3. Discover security concepts used in your house.

My 'house' is a student accomodation so things are slightly different than your traditional home. 

## Elevator Code

In order to use the elevators around the accomodation, a 4 digit code must be entered prior to the lift being called. This is to prevent non-residents from being able to access wings (and hence rooms) without knowledge of the code. This is also changed every year to prevent ex-residents from also accessing the areas the elevator takes you to.

## Bike Room Lock

Due to many thefts in previous years, areas where there are residents' bikes are locked at night between 10pm-8am (as stated on the sign). This is because cameras can only do so much and due to the open-nature of the accomodation, it makes it vulnerable to theft, so this lock can and has helped reduce the risk of bike thefts.

## Wing Locks

Every wing in the accomodation has a lock to access the wing. All residents can unlock each wing's area, but this again, prevents outsiders from being able to access wings and hence resident's rooms, although this is typically not used as resident's tend to lock their doors.

## Alarm System

Rooms which contain valuables (such as the administration building, and piano and art gallery room), have an alarm system that is turned on whenever the room is not in use. These rooms are always locked unless being used but additionally have this alarm system due to the sheer amount of valuable information in these rooms.

## WiFi Password

This might seem extremely basic but this is a security concept. The Wifi requires a password to access, albeit a not so secure one, but a password helps to secure the network from people passing by and protecting personal data. Every few rooms has a separate WAP (wireless access point) as well which from a security perspective, also helps to identify where unauthorised activity is occurring.
